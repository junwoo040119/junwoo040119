# Project Overview

## Title
LLM Planning with Diffusion-style Future Simulation for Multimodal Verification

## Core Loop
Plan -> Simulate -> Verify -> Revise

## Goal
To show that text-only planning can miss physical risks, and that a simulation-verification loop can catch some of those failures.

## Current State
This is a minimal prototype with mock simulation and rule-based verification so that the pipeline can be tested immediately.
